Actions to be performed to launch the game are all carried out in the game directory.

Some antivirus blocks the execution of java, it is necessary to authorize the execution of java to play the game.

If there is not "(32 bit)" in the version name, it use 64 bit java. If your computer is very old you may need a 32 bit java version on your computer if you want to play.

How to use the game ?

Windows : Launch game by double click on formiko.bat !

Mac : Launch game by double click on formiko.sh or with command line : ./formiko.sh

Linux : Before 1a launch : chmod 700 formiko.sh ! Launch game by double click on formiko.sh or with command line : ./formiko.sh !
