Need help ?

Windows
  1. Wait for the download to complete
  2. Validate that you want to keep the file
  3. Open your downloads directory
  4. Double click on the Kokcinelo
  5. Check accept the license
  6. Click install & allow to install with administrator permission
Linux
  1. To install the game launch in a terminal (Replace Download by your download folder if it's different):
    sudo apt install ~/Download/Kokcinelo.deb
  2. To launch game after installation launch in a terminal:
    kokcinelo

If your still have installation issues ask for help on our Discord.
